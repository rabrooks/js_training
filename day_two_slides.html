Problem Five

Write a function that takes a binary function, and makes it callable with two invocations.


addf = applyf(add);
addf(3)(4) => 7

applyf(mul)(5)(6)  => 30

Answer Five


function applyf(binary) {
    return function (x) {
        return function (y) {
            return binary(x,y);
        };
    };
}

Problem Six

Write a function that takes a function and an argument, and returns a function that can supply a second argument.


add3 = curry(add, 3);
add3(4)  => 7

curry(mul, 5)(6);

Answer Six


function curry(binary, x){
    return function (y) {
        return binary(x,y);
    };
}

This is sometimes called the Curry function or schonfinkelisation

Extra Credit


function curry( func, first) {
    return applyf(func)(first);
}

Problem Nine

Write demethodize, a function that converts a method to a binary function


Note: you will need to use call to invoke your return function

Answer Nine


function demthodize(func) {
    return function (that, y) {
        return func.call(that,y);
    };
}

Problem Thirteen

Write a function that allows another function to only be called once


add_once = once(add)
add_once(3,4)  => 7
add_once(3,4)  => throw!

Answer Thirteen


function once(func) {
    return function () {
        var f = func;
        func = null;
        return f.apply(this, arguments);
    };
}

Problem Fourteen

Write a factory function that returns two functions that implement an up/down counter


counter = counterf(10);
counter.inc()  => 11;
counter.dev()  => 10;

Answer Fourteen


function counterf(value) {
    return{
        inc: function () {
            value +=1;
            return value;
        },
        dec: function () {
            value -=1;
            return value;
        }
    };
}

Problem Fifteen

Make a revocable function that takes a nice function, and returns a revoke function that denies access to the nice function, and an invoke function that can invoke the nice function until it is revoked


temp = revocable(alert);
temp.invoke(7);  => Alert: 7
temp.revoke();
temp.invoke(8);  = Alert: 8

Answer Fifteen


function revocable(nice) {
    return {
        invoke: function () {
            return nice.apply(this, arguments);
        },
        revoke: function () {
            nice = null;
        }
    };
}
